### Installation

1. Get API keys from supported AI services

2. Clone the repo

```sh
git clone https://github.com/donaldturinglee/copilotbox-app.git
```
3. Install NPM packages

```sh
npm install
```
4. Run the application
   
Android: 
```sh
npm run start
npm run android
```
iOS:

```sh
npm run start
npm run ios
```
